## Ознакомление
Данный репозиторий предназначен как библиотека материала курса и быстрый и удобный способ получение файла. Также возможно клонировать репозиторий и обновлять при каждом изменение, которое будет оглошаться в канале Файлы колледжа(пока что в ручную). Также приветсвуется добавление нового файла с помощью Pull Request.
